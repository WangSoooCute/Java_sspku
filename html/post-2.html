HTML、CSS、JavaScript

HTML、CSS、JavaScript是前端开发的基础，分别用来描述网页的结构、样式和行为。
这3个文件可以写在同一个文件中，也可以分开写。

一个简单的例子：

在同一个文件夹内新建3个文件：index.html、styles.css、script.js

index.html： 


<!DOCTYPE html>
<head>
    <meta charset="UTF-8">
    <title>基本示例</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>

styles.css： 


/* 设置body样式 */
body {
    text-align: center; /* 居中对齐 */
    font-family: Arial, sans-serif; /* 设置字体 */
    background-color: #f0f0f0; /* 背景颜色 */
    margin: 0; /* 去掉默认边距 */
    padding-top: 20%; /* 页面顶部间距 */
}

/* 设置h1样式 */
h1 {
    color: #333; /* 标题颜色 */
}

/* 设置body样式 */
button {
    padding: 10px 20px; /* 按钮内边距 */
    font-size: 16px; /* 按钮文字大小 */
    background-color: #007bff; /* 按钮背景颜色 */
    color: #fff; /* 按钮文字颜色 */
    border: none; /* 去掉边框 */
    cursor: pointer; /* 鼠标指针效果 */
}

/* 按钮悬停效果：鼠标悬停时背景颜色变深 */
/* button:hover {
    background-color: #0056b3;
} */

script.js： 


//给按钮添加点击事件监听器
document.getElementById('changeButton').addEventListener('click', () => {
    const title = document.getElementById('title'); //获取标题元素
    title.textContent = 'You clicked the button!'; //修改标题的文字内容
    title.style.color = '#007bff'; //修改标题的颜色
});

    <!-- 导入 vue.js 的 script 脚本文件 -->
    <script src="https://unpkg.com/vue@3"></script>
</head>

<body>
    <!-- 在页面中声明一个将要被 vue 所控制的 DOM 区域，既MVVM中的View -->
    <div id="app">
        {{message}}
    </div>
    <!-- 创建一个 Vue 实例，并将其挂载到 id 为 app 的 DOM 区域上 -->
    <script>
        Vue.createApp({
                data() {
                    return {
                        message: 'Hello Vue!'
                    }
                }
            }).mount('#app')
            // 也可以分开写：

            // }

Axios网络请求

Axios用于在浏览器端使用XMLHttpRequests发送网络请求。
官方网址： https://www.axios-http.cn/

打开刚刚创建的component-demo项目，并安装Axios：


npm install axios

打开DataGrip，让数据库保持运行状态。
打开我们上次创建的mpdemo项目，并把端口改为自定义的8088： 


server.port=8088

在UserController.java中加入可以允许跨域的注解： 


@CrossOrigin

在App.vue文件的对应位置添加以下代码： 


import axios from 'axios'//写在<script>里面

created:function() {//写在export default里面
    axios.get('http://localhost:8088/user').then(response => {
      console.log(response.data)
    })
},

在浏览器中进入开发者模式，并查看Network部分，可以看到请求的结果：

让Axios获取的后端数据显示在前端页面上：

把以下代码复制粘贴到Hello.vue文件的对应位置： 


import axios from 'axios'//写在<script>里面

created:function() {//写在export default里面
    axios.get('http://localhost:8088/user').then(response => {
      this.tableData=response.data
    })
},

把tableData清空：
